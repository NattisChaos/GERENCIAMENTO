# GERENCIAMENTO

Projeto para gerenciar cadastros de alunos em clubes e cursos.

## Objetivo

Organizar e armazenar informações dos alunos para facilitar contato, atendimento e gestão de turmas.

## Funcionalidades (exemplo)

- Cadastro de alunos com campos como nome, data de nascimento/idade, telefone, telefone familiar, alergias, condições especiais e observações.
- Busca, edição e remoção de registros.
- Agrupamento por turma/curso e exportação de listas.

## Estrutura de dados

- Nome completo
- Data de nascimento / Idade
- Telefone principal
- Telefone familiar
- Alergias / Condições especiais
- Observações
